Windows

La primera versión de Windows fue la 1.0 lanzada en 1985 para competir con el sistema operativo de apple, no fue muy popular el cual era una extensión grafica de ms dos (MicroSoft DiskOperatingSystem). Windows 2.0 en 1987 incluyó la caracteristica de que las ventanas (a partir de ventanas de operación) podian sobreponer unas a otras.

Windows Wikipedia
Microsoft

El logo de Windows hace alución a una ventana del mundo fisico debido a que fueron los primeros en implementar el concepto de ventanas de operación en el cual se podian mantener dos programas en simultaneo. La marca ha cambiado de 4 colores con un diseño curvo a un diseño mas cuadriculado.

Ventajas

  1. Muchas personas y empresas lo usan
  2. Trabajar software que solo existe para mac o windows como la suite de Adove, juegos, editores de audio entre otras cosas
  3. Acuerdos comperciales con casas de software para drivers

Desventajas

  1. La instalacion y actualización de software no es muy eficiente
  2. El trabajo desde la terminal no es muy comodo
  3. Consume una gran cantidad de recursos
  4. Las actualizaciónes del S.O tienen a ser involuntarias e inoportunas.
  5. No existen actualizaciónes de drivers para cierto hardware
  6. Es un software que se adquiere mediante licencias pagadas
