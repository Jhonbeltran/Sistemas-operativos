¿Que es un sistema Operativo?

Un sistema operativo (s.o) es un conjunto de servicios (programas) los cuales hacen de puente entr el usuario que se comunica en lenguaje natural y la maquina que entiende lenguaje binario.

¿Que sucede en el proceso de inicio?

  1. Presionamos el boton de encendido cerrando el circuito entre la fuente y la placa base.
  2. Cuando la placa base esta siendo alimentada le envia una indicación a la Bios para que inicie.
  3. La Bios hace un check de todos los perifericos conectados en donde busca los controladores y comprueba que todo esté funcionando.
  4. La Bios le envia un mensaje a la memoria RAM para que se comunique con el disco duro.
  5. La RAM A la indicacion para que el disco duro inicie en su Zona 0 (boot.init).
  6. El disco duro carga los archivos necesarios a la memoria ram y pide que se ejecute el sistema operativo (system.init).
  7. El sistema inicia y va a estar listo hasta que la barra de tareas inicie (antes de eso va a estar cargando los servicios).
