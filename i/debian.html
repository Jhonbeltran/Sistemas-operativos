Continuando nuestro proceso de instalación de dos sistemas operativos en una misma maquina vamos a instalar una distribución de GNU/Linux (Debian 8) la cual nos va a instalar el Grub para poder arrancar el sistema operativo que querramos al iniciar nuestra maquina. Los pasos son:

1. Iniciar el instalador grafico

Vamos a cambiar el cd de arranque en la maquina virtual por el de debian, luego seleccionamos la opcion Graphical Install.

2. Seleccionamos el Idioma

El idioma que definamos va a ser usado en el proceso de instalación y será el lenguaje predetermiando de nuestro sistema operativo.

3. Seleccionamos el Pais

La lista de paises estará definida por el lenguaje que escojamos en el anterior paso.

4. Seleccionamos el teclado

De acuerdo a la configuración del teclado que tiene nuestra pc buscamos la que mejor se le adapte.

5. Introducimos el nombre del equipo

Este nombre aparecerá en nuestra terminal como el superUsuario user@superuser.

6. Creamos una contraseña superuser

La contraseña de super usuario la vamos a requerir cada que querramos instalar algun programa, actualizar el sistema, entre otras cosas.

7. Agregamos nuestro nombre completo

Agregarlo nos ayudará a tener una mayor personalización de nuestra cuenta, asi como una mejor integracion con los servicios que nos provee debian.

8. Creamos el nombre del usuario

Este nombre va a ser usado por la cuenta normal del sistema, cuando se requieran permisos nos va a solicitar la contraseña del superusuario.

9. Creamos el nombre del usuario

Este nombre va a ser usado por la cuenta normal del sistema (la cual tambien va a requerir contraseña), cuando se requieran permisos especiales nos va a solicitar la contraseña del superusuario.

10. Definimos el particionado de disco

Justo despues de que el asistente de instalación configure el reloj aparecerá el modulo de particionado de disco en donde escogeremos el modo manual. definiremos 3 particiones fisicas en el espacio libre que teniamos.

solo tengan en cuenta estas 3 las demas que se ven en la siguiente imagen sin irrelevantes en este punto.

Estas son las particiones que ordené para este ejemplo
  • - Una particion Swap como area de intercambio
  • - Una particion / (root) en formato ext4 en donde instalaremos el sistema operativo la cual marcaremos como particion de arranque
  • - Una particion /home en donde almacenaremos nuestros archivos como multimedia entre otros
11. Instalacion del sistema

En este paso el asistente de instalación va a copiar los datos al disco duro para iniciar con la instalacion. Los archivos de instalacion se van a ejecutar directamente desde el disco duro. Luego de que se copien los archivos el programa va a instalar carcador de arranque GRUB. El cual va detectar el sistema operativo que tengamos instalado previamente para que nos permita iniciar con el so que queramos.

12. Instalación del cargador de arranque

En este punto seleccionaremos el disco que esta usando la maquina virtual.

13. Seleccionamos que os iniciar

Usando el GRUB al iniciar nuestra maquina luego de que la instalación termine podremos escoger que sistema operativo arrancaremos.

14. Hemos terminado

Enhorabuena!.
